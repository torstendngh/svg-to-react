# SVG → React

Paste an SVG and instantly get a typed React component.

## Features

- TSX or JSX output
- Replaces hardcoded colors with `currentColor`
- Copy or download the generated file

## Dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000), paste your SVG on the left, get your component on the right.
